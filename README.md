🏥 LifeLine Hospital Management System

A simple and user-friendly Hospital Management System built using **Node.js, Express.js, HTML, and CSS**. The project allows hospital staff to register patients, store their details, and view patient records through a web interface.

---

📌 Features

* Patient admission form
* Store patient details in a text file
* View all registered patients
* Responsive and modern user interface
* Professional hospital-themed design
* Input validation for required fields
* Success confirmation page after registration
* Emergency contact section

---

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Node.js)
* Express.js
* File System Module (fs)
* npm (Node Package Manager)

🧪 How to Use

1. Open the application in your browser.
2. Fill in the patient admission form.
3. Click **Admit Patient**.
4. The patient information will be stored in `patient_registry.txt`.
5. Click **View Patient Records** to see all registered patients.

---

📋 Patient Information Stored

The system stores:

* Patient Name
* Age
* Gender
* Date of Birth
* Phone Number
* Address
* Disease/Illness
* Registration Date and Time
